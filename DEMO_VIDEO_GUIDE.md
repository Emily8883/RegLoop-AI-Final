# RegLoop AI Demo Video Guide

## Video Goal

Record a 3 to 5 minute walkthrough of the verified challenge flow:

`Upload -> Extract -> Gap Analysis -> Policy PR -> Human Review -> Audit Trail -> Export`

## Best Recording Strategy

Use the already populated challenge data if you want the cleanest recording.

The demo is strongest when you show:

- the workspace already populated
- obligations extracted
- compliance gaps created
- one PR created and approved
- audit trail refreshed
- export working

## Suggested Script

### 0:00 - 0:15 Opening

Show the dashboard and say:

"RegLoop AI turns a regulatory update into obligations, gaps, reviewable policy actions, audit evidence, and exportable compliance output."

### 0:15 - 0:50 Documents

Show the Documents workspace with:

- DORA regulation
- three policy PDFs
- responsibility matrix

Say:

"This workspace collects the required inputs in one place. The analysis only starts when the full set is present."

### 0:50 - 1:20 Obligations

Open Obligations and show extracted rows.

Say:

"RegLoop extracts structured obligations from the uploaded regulation and stores traceable compliance data for review."

### 1:20 - 2:00 Compliance

Open Compliance and show the red/high-risk gaps.

Say:

"The system generates gap records and coverage views so the team can prioritize remediation."

Create one PR if you want a live action.

### 2:00 - 2:45 Policy Review

Open Policy Review and show the created PR.

Approve it.

Say:

"Policy recommendations are human-reviewed. AI proposes changes, but a reviewer decides whether to approve, reject, modify, or escalate."

### 2:45 - 3:30 Audit Trail

Open Audit Trail for the analyzed regulation.

Show:

- document upload event
- obligation extraction
- gap assessment
- PR generation
- approval event

Say:

"The audit trail captures the full compliance history for the selected regulation."

### 3:30 - 4:00 Export

Show JSON and CSV export.

Say:

"The full compliance package can be exported for reporting and submission use."

## What To Avoid

- Do not open unrelated old documents
- Do not show terminal windows
- Do not spend too long on file upload
- Do not narrate the old sample-text workflow

## Recording Settings

- Resolution: 1920x1080
- Frame rate: 30fps
- Browser zoom: 125%
- Audio: clear microphone

## Final Checklist

- backend running
- frontend running
- sample data visible
- obligations extracted
- gaps created
- PR approved
- audit trail refreshed
- export working
